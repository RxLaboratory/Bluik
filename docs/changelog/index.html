This is the list of what has changed since the first version of Duik for Blender.

0.5.2

New

  • Added an operator to reset Custom UI Controls to their default values.

0.5.0

New

  • Dope Sheet and Graph Editor filters

Bugfixes

  • Improved TexAnim and fixed a few bugs.

0.4.0

New

  • Added the OCA Importer

0.2.0

New

  • Camera Linker

0.0.20

Bugfixes

  • The TexAnim now updates correctly the textures when are nested inside node groups.
  • Fixed textures which could not be moved or removed from TexAnim when it’s nested inside node groups.

0.0.18

Bugfixes

  • The TexAnim now works correctly when the textures are nested inside node groups.

0.0.17

Bugfixes

  • The TexAnim keyframes are now correctly updated when modifying the list of images if the list already has keyframes.
  • The TexAnim is now correctly updated when navigating the timeline with the keyboard shortcuts ( [->] and [<-] )

New

  • Pie menu with animation tools.
  • Added a Pie menu for the armature display options.
  • Added a Pie menu for the auto-rig tools.
  • Automatically swap IK and FK.

Bugfixes

  • Lots of bugfixes

0.0.6

New

  • Added the Texture Animation tool.

Bugfixes

  • Lots of bugfixes

0.0.5

New

  • Added a ui for bone layers in the 3D View

0.0.4

Source code Improvement and cleaning

As the addon becomes bigger, there was a need to clean the source code to improve the groundwork of future features.

  • Split the addon in modules and submodules for easier development and maintenance
  • Added GPLv3 notices

0.0.3

Improvements

  • FK Controller: the property for the “follow” option has been moved to the bone instead of the controller. Use a Duik UI Control to expose it on the controller if needed. Also, the way to rig the FK controller has been simplified a lot.

0.0.2

New

  • UI Controls

Improvements

  • BBone Controllers now control the bbone rotations more accurately, and the scale controls the curvature.

Fixes

  • The Assign button for the Selection sets now work correctly even if the set has been created empty at first.
  • The custom properties used by Duik no longer have wrong limits.

0.0.1

Initial release with:

  • Rigging tools:
    • IK/FK rig
    • FK Controller
    • FK Controller (No Follow)
    • BBone Controllers
  • UI Controls:
    • Selection sets
